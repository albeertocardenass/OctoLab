¿Qué ofrece OctolabWeb?

🧪

Laboratorios Prácticos

Retos y ejercicios reales para afianzar conceptos de seguridad informática en un entorno controlado.

📚

Temario Interactivo

Módulos estructurados que cubren desde criptografía hasta hacking ético, con contenido PDF integrado.

🏆

Sistema de Puntos

Gamificación real: gana puntos completando módulos y retos para escalar en el ranking de la comunidad.

👥

Comunidad

Foro integrado donde los usuarios pueden compartir dudas, recursos y avances con el resto de la comunidad.

Stack Tecnológico

Frontend Angular 17 · TypeScript · SSR
Backend ASP.NET Core · JWT · EF Core
Base de Datos SQLite · Ligero y portable
Pagos Stripe · Donaciones seguras
