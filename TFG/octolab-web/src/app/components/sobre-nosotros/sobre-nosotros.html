OctolabWeb

Plataforma educativa de ciberseguridad con laboratorios prácticos, temario interactivo y sistema de puntos.

¿Qué es OctolabWeb?

OctolabWeb es un ecosistema completo de aprendizaje en ciberseguridad. Combina una plataforma web, un backend robusto, una aplicación de escritorio y contenedores Docker para ofrecer laboratorios reales donde el alumno practica hacking ético de forma segura y guiada.

🌐

Aplicación Web

Angular 17 SSR
  • Autenticación con roles: Admin, Usuario e Invitado
  • Panel de administración con gestión de usuarios
  • Feed de noticias en tiempo real desde The Hacker News (RSS)
  • Comunidad con posts, respuestas en hilo y likes
  • Sistema de donaciones integrado con Stripe
  • Temario con módulos PDF, puntos y códigos hexadecimales
  • Dark mode completo y diseño responsive
⚙️

Backend

ASP.NET Core + SQLite
  • API REST con autenticación JWT y control de roles
  • Proxy de imágenes para el feed RSS
  • Endpoints de verificación de códigos hexadecimales
  • Tracking de última conexión y progreso por usuario
  • Integración con Stripe para pagos seguros
  • Gestión de módulos PDF protegidos por rol
🖥️

App Desktop

Python + CustomTkinter
  • Login conectado al backend con JWT
  • Verificación automática de Docker en el sistema
  • Lanzamiento de labs: Kali Linux + Metasploitable 2
  • Terminal embebida para acceder a las VMs
  • Temario con tests interactivos por módulo
  • Generación de códigos hex únicos por usuario y tema
  • Distribuida como .exe con PyInstaller
🐳

Infraestructura

Docker + AWS
  • Contenedores Docker con red NAT privada aislada
  • Kali Linux y Metasploitable 2 preconfigurados
  • Despliegue en AWS: EC2, S3, RDS y CloudFront
  • Backend y frontend desplegados en entorno cloud
